# Manager Hierarchy -- Structure

4 tiers: T1 Director -> T2 Managers -> T3 Managers -> Workers.
- Maximum 5 children per manager at every tier -- no exceptions
- T1: 1 director supervises 5 T2 managers
- T2: 5 division managers, each supervises 5 T3 managers
- T3: 25 team managers, each supervises up to 20 workers
- Total fleet: ~500 workers
